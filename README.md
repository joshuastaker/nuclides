## Chart of Nuclides Experiments
- Just playing with the chart of nuclides
- API data from: https://nds.iaea.org/dataexplorer/api_manual

#### Jupyter Notebook (under notebooks/)
Relies on nuclides.db SQLite3 database. To run the Jupyter notebook independently:
- Create SQLite database at: data/nuclides.db
- Run scripts/load_nuclide_data.py
- This will load the nuclide data from the IAEA database
